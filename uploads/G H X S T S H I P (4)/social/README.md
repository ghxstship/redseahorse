## Recipe (make your own)
1. Pick a ground: `--void` (default), `--nebula` (loud announcement), or `--brass` (crew/education).
2. Brand bar top-left: skull mark + "GHXSTSHIP" in Big Shoulders 900.
3. One idea, set huge in display caps; one pop accent word.
4. Mono `◆ Eyebrow` + a manifest stamp (`DATE // CITY`).
5. Optional: POW burst for a stat, pixel chip for a label, hard-shadow button for the CTA.
6. Never more than one signal color per post (plus brass).
