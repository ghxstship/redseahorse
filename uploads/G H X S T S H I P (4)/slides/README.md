# Slides — GHXSTSHIP Pitch Deck Template

A 1920×1080 HTML deck built on the "Venture Beyond" system. **Open `index.html`** and use ←/→ to navigate.
Powered by `deck-stage.js` (scaling, thumbnail rail, speaker-notes hook, print-to-PDF).

## Slide types included (fork these)
1. **Title** — wordmark + white skull in a brass sticker frame, Miami/NY/CHI/LA stamp.
2. **Three Verticals** — Production / Operations / Technology.
3. **Statement** — full nebula panel ("We redefine what's possible").
4. **Stats** — 3-up stat boxes (14+ Years / 250+ Experiences / 5M+ Memories).
5. **Destinations** — Festivals / Concerts &amp; Tours / Brand Activations / Immersive / Sporting.
6. **Flight Path** — the 8-Phase Production Lifecycle (Discovery → Strike).
7. **Technology** — the instruments: ATLVS (Atlas) / COMPVSS (Compass) / GVTEWAY (Gateway).
8. **Case study** — split media / info layout.
9. **CTA** — "Venture Beyond" splash.

## Editing
- Each slide is a static `<section>` — click-to-edit text works in the editor.
- Don't set position/size on the `<section>`s; `deck-stage` positions them.
- Add speaker notes only on request (see the Speaker notes skill).
- Export to PPTX or PDF via the export skills; the deck prints one page per slide.
